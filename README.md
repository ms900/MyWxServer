# MyWxServer  
一、基于SpringBoot的微信公众号后端开发  

二、运行环境：IDEA、jdk8、微信公众号平台、ngrok（https://ngrok.com/）   
  
   
三、项目搭建：创建SpringBoot项目，导入项目；启动ngrok(ngrok http 8080) ，申请微信公众号（https://mp.weixin.qq.com/）
